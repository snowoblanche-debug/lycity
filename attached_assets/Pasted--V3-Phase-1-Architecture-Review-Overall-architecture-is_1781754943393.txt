## 2. Song Session

Approved.

This is an important feature.

---

### Session Title

Default:

YYYY-MM-DD 歌回

Examples:

2026-06-18 歌回

2026-07-05 朝活羽你

Streamer can edit manually.

---

### Session End Behavior

Current proposal:

Auto-complete all playing songs.

Rejected.

---

Use:

If any queue item is currently playing:

Show confirmation dialog.

Example:

"There is currently a song being performed.

Please complete or skip it before ending the session."

Do not automatically complete songs.

Reason:

Automatic completion would create incorrect history records.

---
