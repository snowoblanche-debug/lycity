# LY.city V3.0 - Multi Streamer & Lyrics System

The current system is working well for a single streamer.

I now want to evolve LY.city into a platform that can be shared with multiple VTubers and streamers.

The goal is:

* Easy to learn
* Easy to maintain
* Streamer-first workflow
* OBS integration
* Multi-user support

---

# PRIORITY 1 — LRC / Dynamic Lyrics System

## Goal

Allow streamers to display synchronized lyrics in OBS.

---

## Song Database

Add fields:

songs

* lyrics_text
* lyrics_source
* lyrics_updated_at

---

## Lyrics Search

Add button:

🔍 Search Lyrics

Workflow:

Song Title + Artist
↓
Search online lyrics providers
↓
Preview lyrics
↓
Import into database

Examples:

Aimer カタオモイ

SHE Ring Ring Ring

鄧紫棋 光年之外

---

## Supported Formats

LRC

KRC

YRC

Plain text

Store original format.

## Lyrics Editor

Admin should be able to:

* Edit lyrics manually

Add:

per song.

Examples:

0 days

7 days

14 days

30 days

---

When requesting:

If still in cooldown:

Show warning.

Do NOT block by default.

Allow
or
Block
